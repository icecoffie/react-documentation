Pengantar

Dokumentasi ini membahas bagaimana cara menggunakan React untuk mengambil data dari API menggunakan Axios, serta bagaimana TypeScript membantu membuat kode lebih aman, rapi, dan mudah dikembangkan.

Instalasi Axios

Pasang Axios dengan perintah berikut:

npm install axios
npm install --save-dev @types/axios
Dasar React + API

Contoh mengambil data API dengan React:

import React, { useEffect, useState } from 'react';
import axios from 'axios';

const App = () => {
  const [data, setData] = useState([]);

  useEffect(() => {
    axios.get('https://api.example.com/items')
      .then(res => setData(res.data))
      .catch(err => console.error(err));
  }, []);

  return (<div>
    {data.map(item => <p>{item.title}</p>)}
  );
};
Tampilkan Gambar

Jika API mengembalikan URL gambar:

<img src={item.image_url} alt="Gambar Item" />
Tampilkan Angka

Contoh menampilkan angka statistik:

<p>Total Produk: {data.length}</p>
Tipe Objek (TypeScript)

Gunakan tipe agar data lebih jelas dan aman:

interface Product {
  id: number;
  title: string;
  price: number;
  image_url: string;
}

const [products, setProducts] = useState<Product[]>([]);
JS ke TS

Contoh konversi React JavaScript ke TypeScript:

// JavaScript:
const [data, setData] = useState([]);

// TypeScript:
const [data, setData] = useState<MyType[]>([]);
Deep Dive & Tips
  • Gunakan try-catch dengan async/await untuk penanganan error yang lebih rapi.
  • Pisahkan fungsi fetch ke file service/api.ts.
  • Gunakan React Query atau SWR untuk fetch API yang lebih canggih.
  • Gunakan .env untuk menyimpan BASE_URL API.
